# RS-31_Savchenko_Oleksii_RTOS_LR8

Номер групи: РС-31

Автор: Савченко Олексій

ОПЕРАЦІЙНІ СИСТЕМИ РЕАЛЬНОГО ЧАСУ ТА 32 БІТНІ МК Лабораторна робота 8. Дослідження процесів передачі повідомлень і взаємодії між потоками у операційній системі на прикладі FreeRTOS.

Обладнання: STM32F405, CubeIDE.

Мета: Ознайомитися з механізмами міжпоточної взаємодії у FreeRTOS, навчитися створювати кілька потоків, організовувати передачу повідомлень між ними та реалізувати керування потоками за допомогою кнопок.

Завдання до лабораторної роботи:

Варіант 4. Реалізувати чотири потоки для PD12–PD15, які взаємодіють між собою: натискання кнопки PA0 надсилає повідомлення у потік PD12, який передає сигнал далі у потік PD13, і так по ланцюгу.

Функціонал:

LED1 Task: Керує світлодіодом PE3, отримує повідомлення з черги від кнопки BUT1 (PC13), після отримання передає сигнал далі у чергу LED2 Task.

LED2 Task: Керує світлодіодом PE4, отримує повідомлення з черги від LED1 Task, після отримання передає сигнал далі у чергу LED3 Task.

LED3 Task: Керує світлодіодом PE5, отримує повідомлення з черги від LED2 Task, після отримання передає сигнал далі у чергу LED4 Task.

LED4 Task: Керує світлодіодом PE6, отримує повідомлення з черги від LED3 Task, спалахує та завершує ланцюг передачі.

BUT1 Task: Зчитує стан кнопки PC13, при натисканні надсилає повідомлення у чергу LED1 Task для запуску ланцюгової реакції між потоками.

Підключення:

LED1: GND → LED → резистор 220 Ом → PС10

LED2: GND → LED → резистор 220 Ом → PС11

LED3: GND → LED → резистор 220 Ом → PС12

LED4: GND → LED → резистор 220 Ом → PD2

BUT1: GND → Button → резистор 220 Ом → PC13
